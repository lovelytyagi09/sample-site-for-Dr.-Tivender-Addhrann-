Quick Edit Guide

1. Pick a page from the sidebar.
2. Edit any text field or upload a new image.
3. Hit Save Changes — data saves to Firebase Firestore.
4. Images upload to Firebase Storage and the URL updates automatically.
5. Your site HTML files need to fetch from Firestore on load — see integration note below.

⚠️ Integration Note: This dashboard saves data to Firebase. To make the live site reflect changes, each HTML page must include a small JS snippet that fetches the Firestore document on load and updates the DOM. Ask your developer to add fetchPageContent('home') to each page.
